# 📊 Finance Loan Report Dashboard

## 📌 Project Overview
This project is an interactive Loan Analytics Dashboard built using Power BI.  
The goal of this project is to analyze loan data and generate insights related to loan performance, customer behavior, and risk assessment.

---

## 🎯 Objectives
- Analyze total loan applications and funding
- Identify good vs bad loans
- Understand customer profile (employment, home ownership)
- Track loan trends over time
- Provide data-driven insights for decision making

---

## 🛠️ Tools & Technologies Used
- Power BI → Dashboard creation & visualization  
- SQL → Data extraction and query-based analysis  
- Excel → Data cleaning and preprocessing  
- DAX → Calculations and KPIs  

---

## 📂 Dataset Information
The dataset contains loan-related information such as:
- Loan amount
- Interest rate
- Loan status
- Employment length
- Loan purpose
- Home ownership

---

## 📊 Dashboard Features
- Total Loan Applications Analysis  
- Total Funded & Received Amount  
- Average Interest Rate & DTI  
- Good Loan vs Due Loan Analysis  
- Monthly Trends (MTD, MoM)  
- Loan distribution by purpose, term, and employment length  
- Interactive filters for better analysis

## 💡 Key Insights
- Majority of loans are for debt consolidation and credit card usage  
- Most applicants have 10+ years of employment  
- Around 86% loans are classified as good loans  
- Loan applications show consistent monthly trends
